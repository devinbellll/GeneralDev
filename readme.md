# Install

1. Run `install.sh <profile name>` to install everything
2. Install https://github.com/Karmenzind/monaco-nerd-fonts for nicer tmux symbols
3. Configure `iterm` to use load custom preferences file in `Preferences->General->Preferences` => `GeneralDev/mac/profiles/<PROFILE>`
4. Run `:PlugInstall` in nvim to install plugins
5. If ncm2 errors arise install pynvim with: `pip install pynvim`
